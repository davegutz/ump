3 Install Video Cards

The minimum recommended computer architecture is Pentium II, 350 MHz, 128 MB ram. Anything less will set the user up for a frustrating experience with dropped frames due to slow operation.

  • Install the PCI Brooktree 84x video cards. Follow the instructions that come with the card. Linux (RedHat 6.1) will recognize the cards but will not automatically load the bttv drivers.
  • Edit /etc/rc.d/rc.sysinit and add the following lines BEFORE "Now that we": 
              # Attach bttv
          action "Starting tuner..." /sbin/modprobe -k tuner
          action "Starting bttv..." /sbin/modprobe -a bttv
